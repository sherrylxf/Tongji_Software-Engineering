# 宾馆管理信息系统类图（Mermaid格式 - 简化版）

```mermaid
classDiagram
    %% =====================
    %% 核心实体类
    %% =====================
    
    class User {
        -String id
        -String name
        -String pwd
        -String role
        +login()
        +logout()
    }
    
    class Customer {
        -String id
        -String name
        -String phone
        +register()
        +makeReservation()
    }
    
    class Receptionist {
        -String id
        -String name
        -String staffNo
        +processReservation()
        +processCheckIn()
        +processCheckOut()
    }
    
    class Housekeeper {
        -String id
        -String name
        -String staffNo
        +updateRoomStatus()
    }
    
    class Manager {
        -String id
        -String name
        -String staffNo
        +manageRoom()
        +viewReport()
    }
    
    class Admin {
        -String id
        -String name
        -String staffNo
        +manageUser()
        +configSystem()
    }
    
    class Guest {
        -String id
        -String name
        -String idCard
        -String type
        +checkIn()
        +checkOut()
    }
    
    class Room {
        -String id
        -String type
        -String status
        -Double price
        +queryStatus()
        +updateStatus()
    }
    
    class Order {
        -String id
        -Date createDate
        -Date checkInDate
        -Double deposit
        -String status
        +create()
        +modify()
        +cancel()
    }
    
    class Payment {
        -String id
        -Double amount
        -Date date
        +process()
        +refund()
    }

    %% =====================
    %% 核心控制类
    %% =====================
    
    class AuthController {
        +register()
        +login()
        +logout()
    }
    
    class ReservationController {
        +browseRooms()
        +createOrder()
        +modifyOrder()
        +cancelOrder()
        +queryOrder()
    }
    
    class CheckInController {
        +queryOrder()
        +recordGuest()
        +assignRoom()
        +calcCost()
    }
    
    class CheckOutController {
        +queryOrder()
        +calcCost()
        +processPayment()
        +complete()
    }
    
    class RoomController {
        +queryRoom()
        +addRoom()
        +updateRoom()
        +updateStatus()
    }

    %% =====================
    %% 边界类
    %% =====================
    
    class Web {
        +displayInfo()
        +displayForm()
    }
    
    class Reception {
        +displayStatus()
        +displayForm()
    }
    
    class Management {
        +displayManagement()
    }

    %% =====================
    %% 关系定义
    %% =====================
    
    %% 继承（所有员工都继承User）
    Customer --|> User
    Receptionist --|> User
    Housekeeper --|> User
    Manager --|> User
    Admin --|> User
    
    %% 关联
    Order --> Customer
    Order --> Guest
    Order --> Room
    Guest --> Room
    Order --> Payment
    
    %% 控制类依赖实体类
    AuthController ..> User
    ReservationController ..> Customer
    ReservationController ..> Order
    ReservationController ..> Room
    CheckInController ..> Order
    CheckInController ..> Guest
    CheckInController ..> Room
    CheckOutController ..> Order
    CheckOutController ..> Payment
    RoomController ..> Room
    
    %% 边界类依赖控制类
    Web ..> AuthController
    Web ..> ReservationController
    Reception ..> ReservationController
    Reception ..> CheckInController
    Reception ..> CheckOutController
    Reception ..> RoomController
    Management ..> RoomController
```

## 类图说明

本简化类图包含核心的三层架构：
- **实体层**：User、Customer、Receptionist、Housekeeper、Manager、Admin、Guest、Room、Order、Payment
- **控制层**：AuthController、ReservationController、CheckInController、CheckOutController、RoomController
- **边界层**：Web、Reception、Management

详细说明请参考 **类图说明文档-简化版.md**
